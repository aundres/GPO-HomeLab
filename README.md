# GPO-HomeLab

## Objective
Designed and implemented a Group Policy Object (GPO) infrastructure within an Active Directory domain environment. The goal was to practice both centralized Windows administration and security policy enforcement — simulating how IT admins and security teams manage and harden domain-joined systems in enterprise environments.

## Environment
- Windows Server 2022 (Domain Controller)
- Windows 10 (Client VM)
- VirtualBox

## GPOs Created
- Password Policy
- Account Lockout Policy
- Restict Control Panel
- Disable USB Devices
- Desktop Wallpaper
- Drive Mapping
